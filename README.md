# Stash

Share any link to this app, and it auto-organizes it so you can actually find it again.

An Android-first mobile app that saves links from TikTok, Instagram, YouTube, and articles into one organized, searchable place. AI auto-generates a title, summary, and folder category for every saved item.

## Stack

- **Mobile:** React Native + Expo (TypeScript)
- **Backend:** FastAPI (Python) on Render
- **Database:** Supabase (PostgreSQL)
- **AI:** Gemini 3 Flash (Google AI Studio)
